# File Organizer Bot

Скрипт для автоматической сортировки файлов в реальном времени. Он отслеживает изменения в указанных папках и перемещает файлы в нужные директории, основываясь на их расширениях. Начинает работу только после добавления или изменения файлов в указанных директориях.

## Основные возможности
* **Real-time мониторинг**: Использует библиотеку `watchdog` для мгновенной реакции на новые файлы.
* **Многопоточность**: Сортировка и перемещение происходят в отдельном потоке, не блокируя мониторинг.
* **Гибкая настройка**: Все правила перемещения и пути хранятся в YAML формате. Предложенная конфигурация рассчитана на Linux-системы.
* **Предотвращение конфликтов**: Если файл с таким именем уже существует в целевой папке, скрипт автоматически добавит числовой индекс к имени.

## Архитектура
Проект разделен на модули:
* `main.py` — точка входа и инициализация потоков.
* `coordinator.py` — управление наблюдателем (Observer) и событиями файловой системы.
* `sorter.py` — логика сопоставления расширений и путей.
* `mover.py` — безопасное копирование и удаление файлов.

1. Клонируйте репозиторий:

2. Запуск: python main.py
